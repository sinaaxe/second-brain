# Second Brain Knowledge Management System

This repository serves as a personal second brain, designed as a compounding knowledge base maintained in partnership with LLM agents. It features a structured directory system split into `/raw` (incoming clips and notes), `/wiki` (Obsidian-compatible organized markdown entries), and `/archive` (processed history), all managed automatically via customized Gemini CLI skills (Translate, Lint, and Answer Question) defined inside `.agents/skills/`.
